# imitate-bilibili
#### 介绍
模仿实现哔哩哔哩的网站首页和部分功能
#### 安装教程

1.  `git init` 初始仓库
2.  `get pull https://gitee.com/CodeMan-L/imitate-bilibili.git` 获取代码
